# Agent Instructions

This is a macOS dotfiles repository. It stores application configurations under `configuration/` and provides an idempotent `install.sh` to bootstrap a new machine.

## Code Style

- Place comments on the line above the code they describe, not inline at the end.
- Comments should explain **why**, not what — assume the reader can read the code.

## Skills

Skills live in the `skills/` directory at the repo root.

| Skill | Description |
|-------|-------------|
| [add-app-config](skills/add-app-config/SKILL.md) | Add a new app's config to this repo, update install.sh and README |
